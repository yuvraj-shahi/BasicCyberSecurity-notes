# Performance Terms in Computer Networking
 
## Bandwidth 
- Bandwidth means the best performance or total capacity of the network like : 100 mbps of wifi.

## Throughput
- Throughput means the actual real world performance of the network like : only provide 60 mbps.

## Latency
- Latency means the delay time for data to travel from source to destination.
- if we click any website then how much time it take to load the sites is called latency.

## Jitter
- Jitter means the variation or the inconsistency in the latency time.
- if one packet takes 10 ms and other packet take 50 ms to travel then this shows jitter.
