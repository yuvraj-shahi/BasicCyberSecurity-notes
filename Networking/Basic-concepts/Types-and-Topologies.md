# Types of Computer Network

## 1. LAN (Local Area Network)
- The network that connects the computer and devices on local area like home , office etc.

## 2. MAN (Metropolitan Area Network)
- The network that connects the computer and devices on city or town area.
- It connects multiple LAN within a city.

## 3. WAN (Wide Area Network)
- The network that connects the whole world called internet.

## 4. PAN (Personal Area Network)
- The network that connects only someone's personal devices.
- Like : connect my mobile with earphone , bluetooth.

# Types of Topologies

Network Topology means the arrangement of how computer and devices are connected in a network.

## Star Topology
- In this topology computers are connected with a central system (hub or switch) in a star like structure.
- if central device fails then whole network fails .


## Bus Topology
- In this topology computers are connected with a main cable.
- Computers are arranging left and right of the main cable like seats arrangement in bus.
- Data may collide, if a main cable has problem then whole system fails.

## Ring Topology
- In this topology computers are connected in a circular arrangement.
- Each devices are connected with two other devices.
- Harder to troubleshoot , if one devices fails then whole network can stop.
- No data collision problem .

## Mesh Topology
- In this topology computers are connected with each other.
- Every devices are connected with multiple devices.
- Very reliable but expensive.
